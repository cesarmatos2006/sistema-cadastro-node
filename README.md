# ficha de cadastro

## Descrição
Projeto desenvolvido para praticar 
Feito para cadastrar nome, numero, email e data de nascimento
Com funcionalidades para mes e hora, selecionar cor e barra de satisfação,
Vinculo com servidor node e banco de dados

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- Node.js 

## Funcionalidades
- Cadastro de dados
- Listagem
- Interação com usuário

## Como executar
1. Baixe o projeto
2. Abra o arquivo index.html
